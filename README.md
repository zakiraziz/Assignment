## 🚀 Getting Started
To run this website locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/zakiraziz/portfolio-website.git
   ```
2. Navigate to the project folder:
   ```bash
   cd portfolio-website
   ```
3. Open `index.html` in a web browser.
